## SDK usage

::::Tabs
:::Tab{title="JavaScript"}
```javascript
import { Archbee } from "@archbee/sdk";

const client = new Archbee({ token: process.env.ARCHBEE_TOKEN });
const me = await client.me();
console.log(me.email);
```
:::
:::Tab{title="Python"}
```python
from archbee import Archbee

client = Archbee(token=os.environ["ARCHBEE_TOKEN"])
me = client.me()
print(me.email)
```
:::

## Full request/response

::::CodeDrawer{title="Create a document"}
:::CodeblockTabsExamples
```bash
curl -X POST https://api.archbee.com/v2/docs \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"title":"Hello","spaceId":"abc"}'
```
:::
:::CodeblockTabsResponses
```json
{
  "id": "doc_123",
  "title": "Hello",
  "spaceId": "abc"
}
```
